## üîß COMMON ISSUES & SOLUTIONS

### Issue: Filters not appearing
**Solution:**
1. Check filter row exists in HTML
2. Verify CSS is loading
3. Open console (F12) for errors
4. Reload page

### Issue: Data not loading
**Solution:**
1. Check JSON file exists in `/json` folder
2. Verify file names match fetch calls
3. Check network tab (F12) for 404s
4. Validate JSON (jsonlint.com)

### Issue: Add/Edit/Delete not working
**Solution:**
1. Check form validation
2. Verify localStorage working
3. Check file permissions
4. Review console for errors

### Issue: Filters very slow
**Solution:**
1. Increase debounce delay (300ms ‚Üí 500ms)
2. Reduce dataset size
3. Implement pagination
4. Check browser memory usage

### Issue: Responsive layout broken
**Solution:**
1. Check viewport meta tag
2. Verify Bootstrap CSS loaded
3. Check media queries
4. Test different screen sizes
