## ğŸ¨ TABLE FILTERING SYSTEM

### Header-Level Filters
All tables have dynamic filters in the header row:

**Client Table Filters:**
- Text: name, email, phone
- Date: registration date
- Number: visits (min), loyalty points (min)
- Select: preferred contact method

**Services Table Filters:**
- Text: service name, description
- Number: price (min)
- Select: active status

**Appointments Table Filters:**
- Text: appointment ID, client, service, staff
- Date: appointment date
- Select: location (studio/home), status

**Schedule Table Filters:**
- Text: client, service
- Date: appointment date
- Select: location, status

**Features:**
- Real-time filtering (debounced for performance)
- Case-insensitive text search
- Multiple simultaneous filters
- Numeric minimum value filtering
- Date filtering
- Dropdown select filtering
