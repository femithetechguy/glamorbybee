### How Filters Work
All tables now have **header filter rows** with real-time filtering:

```

    â†“

    â†“
Real-time matching (debounced)
    â†“

```

### Filter Types
1. **Text** - Contains search (case-insensitive)
2. **Number** - Exact match
3. **Date** - Date picker format
4. **Select** - Dropdown from data

### Example: Client Table Filters

## ğŸ”§ COMMON TASKS

### Add New Client
1. Go to Client module
2. Click "Add New Client"
3. Fill form: name, email, phone, contact preference
4. System generates ID: CUST0XXX
5. Saves to clients.json

### Create Appointment
1. Go to Appointments module
2. Click "Create New Appointment"
3. Select client, service, staff, date
4. Click Save
5. System auto-generates APT00XXX
6. Auto-creates invoice entry

### View Service Details
1. Go to Services module
2. Switch between Grid/Table view
3. Apply filters to find service
4. Click row to see details

### Check Staff Availability
1. Go to Schedule module
2. View by date
3. See all staff working hours
4. See booked appointments

## ğŸ› TROUBLESHOOTING

### Data Not Loading?
**Check:**
1. JSON files exist in `/json` folder
2. File names match fetch calls
3. JSON is valid (use JSONLint)
4. Browser console for errors (F12)

### Filters Not Working?
**Check:**
1. Header filter row visible (may need scroll)
2. JavaScript not throwing errors
3. Clear filters to reset

### Module Not Showing?
**Check:**
1. Module folder exists
2. index.html file present
3. CSS and JS files linked correctly
4. Local server running (http.server)
